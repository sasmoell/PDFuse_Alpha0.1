DOKUMENTATION

PDFuse Alpha 0.1.2306

Hauptansicht

PDFuse ist ein Programm um PDF-Dokumente zusammenzufügen (Merger) oder zu teilen (Splitter). Es wurde in Python geschrieben und nutzt vor allem die externe Bibliothek pypdf.
Folgend finden Sie die Dokumentation um mit dem Tool umgehen zu können.

Menüleiste - Dateimenü

  1. PDF zusammenzufügen (Fuser)

    2. Startet den Fuser um mehrere PDF-Dokumenten zu einer Datei zusammenzufügen. Der Fuser ist beim Start des Programms sichtbar.

  2. PDF teilen (Splitter)

    3. Startet den Splitter um aus einem PDF-Dokument alle Seiten zu extrahieren und je Seite eine Datei zu erstellen.

  3. Beenden

    4. Beendet das Programm

Menüleiste - Optionenmenü

  1. Ordner anlegen

    2. Legt die erforderlichen Ausgabeordner an. Dadurch werden die Ordner im Stammverzeichnis des Programms angelegt.

    Ordner: output
    Unterordner: mergeoutput
    Unterordner: splits

Menüleiste - Hilfemenü

  1. Info

    2. Zeigt Informationen zum Programm.

  2. Dokumentation

    3. Verweist auf diese Dokumentation.

  3. Check for Updates

    4. Überprüft ob ein Update vorhanden ist. Benötigt eine funktionierende Verbindung ins Internet.

  4. Update Download

    5. Verweist auf die Downloadseite des PDFuser.

Dateien mit PDFuser zusammenfügen

  1. Ordner suchen

    2. Sie haben Ihre Dateien in einem Ordner liegen und möchten diese nun zu einer Datei verschmelzen. Klicken Sie zunächst auf "Ordner suchen" um den Ordner mit Ihren Dateien zu finden.

Datei-Manager öffnet sich

Es öffnet sich der Explorer (Windows) oder der Finder (macOS). Auch wenn der PDFuser ausdrücklich für Windows entwickelt wurde, sollte er mit macOS und Linux kompatibel sein. Getestet wurde er allerdings nur unter Windows 11.

Ordner auswählen

  1. Das richtige Laufwerk wählen

    2. Wählen Sie das Laufwerk aus, indem sich Ihr Ordner befindet.

  2. Den richtigen Ordner wählen

    3. Wählen Sie nun den Ordner aus, in dem sich Ihre PDF-Dateien befinden.

  3. "Ordner auswählen" klicken

    4. Mit "Ordner auswählen" schließen Sie das Fenster wieder und haben den korrekten Ordner ausgewählt.

  1. Pfad zum Ordner

    2. Hier sollte jetzt der korrekte Ordnerpfad angezeigt werden. Wenn dem so ist, kann es weitergehen.

  2. Fuse NOW!

    3. Mit einem Klick auf "Fuse NOW!" wird der Vorgang gestartet.

Es erscheint die Meldung, dass die Datei erstellt wurde.

  1. Ausgabeordner öffnen

    2. Wo sind Ihre Dateien? Sie finden sie im Stammverzeichnis des Programms unter "output". Sie können mit einem Klick auf "Ausgabeordner öffnen" den Dateipfad ganz einfach öffnen.

  1. Ausgabeordner

    2. Der Ausgabeordner des PDFuser ist unter "output" und ist mit "mergeoutput" benannt.

  1. Ausgabe-Datei

    2. Die neu erzeugte Datei (new_file.pdf) beinhaltet alle PDF-Dokumente, die sich im Ursprungsordner befunden haben. Hinweis: Die Dateien befinden sich immer noch dort. Es wurde jeweils eine Kopie erstellt und diese wurden dann zusammengefügt.

  2. Seite 1 von 3

    3. Das Dokument hat nun drei Seiten in diesem Beispiel.

PDF teilen (Splitter)

  1. Startet den Splitter um aus einer PDF-Datei alle Seiten zu extrahieren und zu speichern.

  1. Datei suchen

    2. Öffnet den Windows-Explorer um nach der Datei zu suchen, die gesplittet werden soll.

  1. Datei angeben, die gesplittet werden soll.

  2. Anschließend im Dialog "Öffnen" klicken.

  1. Der Dateipfad wird hier angezeigt.

    2. Sollte der Pfad nicht angezeigt werden, wiederholen Sie den Vorgang.

  2. Splitt NOW!

    3. Startet den Vorgang um alle Seiten zu extrahieren und zu speichern.

Sollte der Ausgabeordner noch nicht vorhanden sein, wird gefragt, ob er erstellt werden soll. Der Ausgabeordner ist notwendig um die Dateien zu speichern. Die Meldung sollte mit "Ja" bestätigt werden.

Wenn die Ordner erstellt werden konnte, wird diese Meldung erscheinen. Diese ist mit "OK" zu bestätigen.

Die nächste Meldung weist darauf hin, dass die Dateien erstellt wurden.

  1. Ausgabe-Ordner öffnen

    2. Die Dateien befinden sich im Ordner "output/splits". Der Ordner kann bequem über den Button aufgerufen werden.

  1. Ausgabe-Ordner

  1. Ausgabe-Dateien

    2. Die neu erzeugten Dateien befinden sich nun im Ausgabe-Ordner.

Menü: Optionen --> Ordner anlegen

Die Option "Ordner anlegen" legt die benötigten Ausgabeordner an. Es sind Schreibrechte im Dateisystem notwendig. Dieser Schritt ist optional, da er bei Bedarf vom Programm auch abgefragt wird. Wenn diese Option genutzt wird, werden im Programmverzeichnis folgende Ordner erstellt:

1. Ordner: output
2. Unterordner: mergeoutput
3. Unterordner: splits

Sollten die Ordner bereits existieren, wird entsprechend diese Meldung ausgegeben.

Sind die Ordner noch nicht vorhanden, wird eine entsprechende Abfrage gestartet.

Ordner für den Merger (zusammenfügen von PDFs) erstellt.

Ordner für den Splitter erstellt

Ausgabeordner

  1. Hilfemenü: Info

    2. Zeigt Informationen zum Programm an wie z.B. Version, Urheberrechte, Lizenz, Kontakt etc.

  1. Dokumentation

    2. Verweist auf diese Dokumentation. Funktionierende Verbindung zum Internet ist für die Online-Version erforderlich.

  1. Check for Updates

    2. Diese Funktion überprüft, ob sich auf dem Server ein Update befindet. Eine Verbindung zum Internet ist zwingend notwendig.

Falls ein Update vorhanden ist, wird diese Meldung ausgegeben.

  1. Update Download

    2. Verweist auf diese Seite. Hier können Updates heruntergeladen werden.

  1. Beenden
  2. Beenden

    3. Beide haben die gleiche Funktion: Sie beenden das laufende Programm
